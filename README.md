# znaki.css ![Travis](https://img.shields.io/travis/nukeop/znaki.css.svg?style=for-the-badge)

## Getting started

To build the css:

```shell
$ ./build.sh
```

This also builds the demo.

To start developing:

```shell
$ ./watch.sh
```

This starts scss in watch mode, updating the css as you edit the scss files.
